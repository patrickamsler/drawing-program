Requires Java 8 or higher

* run tests: `./gradlew test`
* build: `./gradlew build`

Create executable jar and start the drawing program:
1. `./gradlew jar`
2. `java -jar build/libs/drawing_program-1.0-SNAPSHOT.jar`
